Atomic Settings, needed for sites driven by atomic news tools

== Description ==

Atomic Settings allows you to customize you're atomic news tools driven theme and website.

You have the following configurable options

*  Layout settings affect how the site looks.
*  Category settings affect the features of each category (e.g twitter, video..)
*  Category Hopper allow you to customize the atomic-category hopper schedules to suit your site.
* Developer settings provide tools to assist in debug mode
* Header and Sidebar settings allow you to define where categories will show up in the menus.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `AtomicSettings.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure the periodico theme is activated.
4. Ensure Category hopper is activated.
